# 7级精细情绪评分模型

## 概述

将原有的5级情绪标签扩展为7级，每级再细分为3-5个档位，通过deepseek-v4-pro模型进行深度分析，综合多维度因子进行评分。

## 情绪评级体系

### 7级主评级及对应分值

| 评级 | 分值范围 | 说明 |
|------|----------|------|
| 极度恐惧 | -3.0 ~ -2.5 | 市场恐慌性抛售，散户极度悲观 |
| 恐惧 | -2.4 ~ -1.5 | 看空情绪明显，资金流出 |
| 迷茫 | -1.4 ~ -0.5 | 观望情绪浓厚，方向不明 |
| 中性 | -0.4 ~ 0.4 | 多空平衡，情绪稳定 |
| 乐观 | 0.5 ~ 1.4 | 看好后市，资金进场 |
| 积极 | 1.5 ~ 2.4 | 多头情绪强烈，持续上涨 |
| 极度乐观 | 2.5 ~ 3.0 | 市场狂热，散户极度亢奋 |

## 多维度因子分析模型

### 核心因子：市值归一化因子

#### 1. 市值与发帖量比值
```
发帖量因子 = (实际发帖量 / 预期发帖量) * 情绪方向系数

其中：
- 预期发帖量 = 市值 / 10亿 * 基准帖数（如：50帖/10亿市值）
- 情绪方向系数：看多帖占比 - 看空帖占比
```

#### 2. 市值与新闻量比值
```
新闻量因子 = (实际新闻量 / 预期新闻量) * 新闻情绪系数

其中：
- 预期新闻量 = 市值 / 50亿 * 基准新闻数（如：10篇/50亿市值）
- 新闻情绪系数：利好新闻占比 - 利空新闻占比
```

#### 3. 市值与互动量比值
```
互动量因子 = log(1 + 总互动量 / (市值 / 10亿)) * 互动情绪系数

其中：
- 总互动量 = 总回复数 + 总点赞数
- 互动情绪系数：看多互动占比 - 看空互动占比
```

---

### 辅助因子1：行业对比因子

#### 1. 个股 vs 行业情绪差
```
行业对比因子 = 个股情绪分 - 行业情绪分

当：
- > 0.5：个股明显强于行业
- < -0.5：个股明显弱于行业
- 其他：同步行业
```

#### 2. 个股 vs 行业热度差
```
热度对比因子 = (个股发帖量/个股预期发帖量) / (行业发帖量/行业预期发帖量)

结果：
- > 1.5：个股热度异常高
- < 0.5：个股热度异常低
```

---

### 辅助因子2：发帖质量因子

#### 1. 发帖深度评分
```
深度评分 = Σ(单帖字数 / 100) * 情感权重 * 平台权重

其中：
- 情感权重：强烈看多/看空 = 1.5，温和观点 = 1.0，无意义水帖 = 0.2
- 平台权重：雪球 = 1.2，股吧 = 1.0
```

#### 2. 爆值帖占比
```
爆值因子 = 爆值帖数 / 总帖数 * 爆值帖平均情绪分 * 2

注：爆值帖有更高权重
```

#### 3. 热度帖趋势
```
趋势因子 = (最近24小时帖数 * 最近帖平均情绪分) / (前24-48小时帖数 * 前帖平均情绪分)

反映情绪变化趋势
```

---

### 辅助因子3：时间衰减因子

```
时间权重 = e^(-小时数 / 24)

即：
- 0-6小时：权重 1.0 ~ 0.78
- 6-12小时：权重 0.78 ~ 0.61
- 12-24小时：权重 0.61 ~ 0.37
- 24-48小时：权重 0.37 ~ 0.14
- >48小时：权重 < 0.14
```

## DeepSeek-V4-Pro 分析流程

### 阶段1：单帖深度分析

对每一篇帖子/新闻，分析：

1. **情绪方向**：极度看多、看多、中性、看空、极度看空
2. **情绪强度**：0-10分，表达越强烈分数越高
3. **内容质量**：0-10分，是否有实质分析、数据支撑
4. **影响潜力**：0-10分，预估可能引发的讨论热度

输出格式：
```
{
  "post_id": "xxx",
  "emotion_direction": "看多",
  "emotion_intensity": 7.5,
  "content_quality": 6.0,
  "impact_potential": 5.0,
  "raw_score": 7.5 * 6.0 / 10 = 4.5
}
```

---

### 阶段2：市值归一化处理

#### 输入数据
- 股票市值（亿）
- 总发帖数
- 总新闻数
- 总互动数（回复+点赞）

#### 计算预期基准
```
预期发帖数 = max(20, 市值 / 10)
预期新闻数 = max(5, 市值 / 50)
预期互动数 = max(100, 市值 / 2)

说明：设置最低值保证小市值股票也有基础样本
```

#### 计算丰裕系数
```
发帖丰裕系数 = 实际发帖数 / 预期发帖数
新闻丰裕系数 = 实际新闻数 / 预期新闻数
互动丰裕系数 = 总互动数 / 预期互动数

当丰裕系数 > 3 时，按对数压缩：
压缩后系数 = 1 + log(丰裕系数) / log(3)
```

---

### 阶段3：综合评分计算

#### 基础情绪分
```
基础分 = Σ(单帖原始分 * 平台权重 * 时间权重) / Σ(平台权重 * 时间权重)

平台权重：
- 雪球爆值帖：1.5
- 雪球热帖：1.2
- 股吧爆值帖：1.3
- 股吧热帖：1.1
- 普通帖：1.0
- 新闻：0.9
```

#### 市值调整分
```
市值调整分 = 基础分 * (0.5 + 0.5 * min(丰裕系数, 2))

逻辑：
- 当样本量不足时，置信度降低，评分向中性收缩
- 当样本量充足时，保留原始评分
```

#### 行业对比调整
```
行业调整分 = 市值调整分 + (个股分 - 行业分) * 0.3

逻辑：
- 个股显著强于行业时，加分
- 个股显著弱于行业时，减分
- 幅度为差距的30%
```

---

### 阶段4：最终评级映射

将最终分数映射到7级评级：

| 最终分数 | 评级 | 细分档位示例 |
|----------|------|--------------|
| -3.00 | 极度恐惧 | -3.0, -2.9, -2.8, -2.7, -2.6, -2.5 |
| -2.40 ~ -1.50 | 恐惧 | -2.4, -2.2, -2.0, -1.8, -1.6, -1.5 |
| -1.40 ~ -0.50 | 迷茫 | -1.4, -1.2, -1.0, -0.8, -0.6, -0.5 |
| -0.40 ~ 0.40 | 中性 | 精确到小数点后两位 |
| 0.50 ~ 1.40 | 乐观 | 0.5, 0.6, 0.8, 1.0, 1.2, 1.4 |
| 1.50 ~ 2.40 | 积极 | 1.5, 1.6, 1.8, 2.0, 2.2, 2.4 |
| 2.50 ~ 3.00 | 极度乐观 | 2.5, 2.6, 2.7, 2.8, 2.9, 3.0 |

## 数据结构设计

### EmotionScoreV2 数据类
```python
@dataclass
class EmotionScoreV2:
    # 基础评分
    raw_score: float              # 原始评分（未归一化）
    normalized_score: float       # 归一化后评分
    
    # 市值调整
    market_cap_adjusted: float    # 市值调整后评分
    abundance_coefficient: float  # 丰裕系数
    
    # 行业对比
    industry_score: float         # 行业平均情绪分
    industry_adjusted: float      # 行业调整后评分
    
    # 最终结果
    final_score: float            # 最终评分
    rating_level: str             # 7级评级（"极度恐惧"等）
    confidence: float             # 置信度（0-1）
    
    # 详细分析
    factor_breakdown: Dict[str, float]  # 各因子贡献
    key_posts: List[Dict]         # 关键影响帖子
    trend_analysis: str           # 趋势分析文字
```
